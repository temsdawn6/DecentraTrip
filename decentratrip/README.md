# ✈️ DecentraTrip

**DecentraTrip** is a decentralized travel booking platform built on the Stacks blockchain using Clarity smart contracts. It allows users to book trips, buy travel insurance, and leave reviews — all without relying on centralized platforms.

---

## ⚙️ Features

- Book hotels and flights with smart contracts
- Get automatic insurance payouts for delays or cancellations
- Earn loyalty rewards for every booking
- NFT-based booking confirmations
- Leave on-chain reviews
- DAO for disputes and platform upgrades

---

## 🧪 How to Run

### 1. Install Clarinet

```bash
npm install -g @hirosystems/clarinet
```

### 2. Clone and Start

```bash
git clone https://github.com/your-org/decentratrip.git
cd decentratrip
clarinet check
